# Hakim Shaisham Portfolio

A static portfolio website based on GitHub projects from [@Hakimcorleone](https://github.com/Hakimcorleone).

## What is included

- `index.html` - portfolio content and project sections
- `styles.css` - responsive visual design
- `script.js` - small enhancement for the footer year
- `vercel.json` - static Vercel configuration

## Featured projects

- Attendance App - team leave, WFH, half-day, and dashboard workflows
- Bisnes Calculator - private multi-business cash flow and hutang tracker
- OrderTrack - multi-business order recorder with Google Sheets sync
- Taska Iris Website - public website for a childcare centre
- Kuih Vendor Tracker - small business vendor tracking tool
- Credit Risk Scorecard Malaysia - finance and credit risk project

## Deploy on Vercel

1. Import this GitHub repository into Vercel.
2. Framework preset: `Other`.
3. Build command: leave empty.
4. Output directory: leave empty or use `.`.
5. Deploy.

This site has no npm dependencies and can be served directly from the repository root.
